# 💻 Ancelmo Machine

## 📌 Sobre o Projeto
Este projeto foi desenvolvido como parte da disciplina de **Arquitetura Computacional**.

O sistema tem como objetivo principal auxiliar no entendimento de conceitos fundamentais da computação, com foco em:

- 🔢 Conversões numéricas
- 🔤 Tabela ASCII
- ⚙️ Representação de dados no computador

---

## 🚀 Funcionalidades

- Conversão entre diferentes bases numéricas:
  - Binário (base 2)
  - Octal (base 8)
  - Decimal (base 10)
  - Hexadecimal (base 16)
- Conversão entre qualquer uma dessas bases
- Visualização da Tabela ASCII
- Interface simples e intuitiva

---

## 🎯 Objetivo

Facilitar o aprendizado de conceitos básicos de como os computadores representam e manipulam dados, sendo uma ferramenta de apoio para estudantes da área de tecnologia.

---

## 🛠️ Tecnologias Utilizadas

- HTML
- CSS
- JavaScript (ou a linguagem que você estiver usando)
- Banco de dados (se aplicável)

---
